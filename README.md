# UnionConfig

Type-safe configuration management for F# applications. Define your config variables as discriminated unions, then read from environment variables, `.env` files, or AWS SSM Parameter Store with automatic parsing, validation, and secret masking.

```fsharp
type AppConfig =
    | DatabaseUrl of ConfigVarDef
    | ApiKey of ConfigVarDef
    | MaxRetries of ConfigVarDef
    | DebugMode of ConfigVarDef

let databaseUrl =
    DatabaseUrl
        { Name = "DATABASE_URL"
          Kind = Manual
          ValueType = StringType
          Requirement = Required
          IsSecret = false
          Doc = { Description = "PostgreSQL connection string"
                  HowToFind = "Check your database provider dashboard"
                  ManagementUrl = None } }

let dbUrl = Reader.readString (match databaseUrl with DatabaseUrl d -> d)
// Reads DATABASE_URL from environment, fails if missing
```

**What you get:**
- Typed parsing (string, int, bool, float, custom validators)
- Required vs optional with compile-time awareness
- Secret masking for safe logging
- `.env` file diffing to detect config drift
- AWS SSM Parameter Store backend
- Interactive editor workflow for config management

## Installation

```bash
# Core library (types, env reader, .env files, verification)
dotnet add package UnionConfig

# AWS SSM Parameter Store backend
dotnet add package UnionConfig.Ssm

# Interactive text editor workflow (CLI tools)
dotnet add package UnionConfig.TextEditor
```

## How It Works

Configuration variables are defined as `ConfigVarDef` records. Each record captures the variable's name, type, requirement level, whether it's a secret, and documentation metadata. You group related vars using discriminated unions for exhaustive handling.

The `Reader` module reads values from environment variables with automatic type parsing. The `EnvFile` module handles `.env` file I/O, diffing, and secret masking. The `SsmConfigStore` module maps var names to AWS SSM paths for cloud-native config management.

## Packages

| Package | Description |
|---------|-------------|
| `UnionConfig` | Core types, environment reader, `.env` file operations, verification |
| `UnionConfig.Ssm` | AWS SSM Parameter Store backend (get/set/delete/list) |
| `UnionConfig.TextEditor` | Interactive `$EDITOR` workflow for editing config with diff + verification |

## Basic Usage

### Defining Config Variables

```fsharp
open UnionConfig.Types

// Define each config var with its metadata
let dbHost =
    { Name = "DB_HOST"
      Kind = Manual
      ValueType = StringType
      Requirement = Required
      IsSecret = false
      Doc = { Description = "Database hostname"
              HowToFind = "Your database provider's connection info"
              ManagementUrl = None } }

let dbPort =
    { Name = "DB_PORT"
      Kind = Manual
      ValueType = IntType
      Requirement = Optional
      IsSecret = false
      Doc = { Description = "Database port"
              HowToFind = "Usually 5432 for PostgreSQL"
              ManagementUrl = None } }

let apiKey =
    { Name = "API_KEY"
      Kind = Manual
      ValueType = StringType
      Requirement = Required
      IsSecret = true
      Doc = { Description = "External API key"
              HowToFind = "API provider dashboard"
              ManagementUrl = Some (System.Uri "https://dashboard.example.com/keys") } }

let debugMode =
    { Name = "DEBUG_MODE"
      Kind = AutoGenerated (Some "false")
      ValueType = BoolType
      Requirement = Optional
      IsSecret = false
      Doc = { Description = "Enable debug logging"
              HowToFind = "Set to true/1 to enable"
              ManagementUrl = None } }

// Custom type with validation
let logLevel =
    { Name = "LOG_LEVEL"
      Kind = Manual
      ValueType = CustomType("LogLevel", fun s ->
          match s.ToLowerInvariant() with
          | "debug" | "info" | "warn" | "error" -> None  // valid
          | _ -> Some "must be debug, info, warn, or error")
      Requirement = Optional
      IsSecret = false
      Doc = { Description = "Application log level"
              HowToFind = "One of: debug, info, warn, error"
              ManagementUrl = None } }
```

### Reading Config from Environment

```fsharp
open UnionConfig.Reader

// Read typed values (fails on missing required vars)
let host = readString dbHost         // string
let port = readIntOrDefault dbPort 5432  // int with default
let key = readString apiKey          // string (required, fails if missing)
let debug = readBoolOrDefault debugMode false

// Read with full ConfigValue for pattern matching
match read dbHost with
| Some (StringValue s) -> printfn "Host: %s" s
| _ -> printfn "No host configured"

// Validate all required vars at startup
let allVars = [ dbHost; dbPort; apiKey; debugMode ]
let errors = validateRequired allVars
if not (List.isEmpty errors) then
    failwithf "Config errors:\n%s" (String.concat "\n" errors)
```

### Working with .env Files

```fsharp
open UnionConfig.EnvFile

// Read a .env file
let config = readEnvFile ".env.production"
// Map<string, string>: { "DB_HOST" -> "prod-db.example.com"; "API_KEY" -> "sk-abc123"; ... }

// Compare two configs to detect drift
let staging = readEnvFile ".env.staging"
let prod = readEnvFile ".env.production"
let changes = compareConfigs staging prod
// Array of (key, oldValue, newValue) tuples

// Mask secrets for safe display
maskValue "API_KEY" "sk-abc12345"   // "sk-a********"
maskValue "DB_HOST" "localhost"     // "localhost" (not a secret key)

// Display changes with masking
displayChanges changes
// Output:
//   API_KEY: sk-a******** -> sk-n********
//   DB_HOST: staging-db -> prod-db
```

### AWS SSM Parameter Store

```fsharp
open UnionConfig.Ssm.SsmClient
open UnionConfig.Ssm.SsmConfigStore

// Configure the store
let store =
    { Config = { Region = Amazon.RegionEndpoint.USEast1
                 EnsureAuth = fun () -> ensureAwsSsoAuth () }
      PathMapping =
          { ToPath = fun name -> sprintf "/myapp/prod/%s" name
            FromPath = fun path -> path.Split('/') |> Array.last
            PathPrefix = "/myapp/prod" }
      IsSecret = fun name -> name.Contains("SECRET") || name.Contains("KEY") }

// CRUD operations
let dbHost = getValue store "DB_HOST"           // string option
let success = setValue store "DB_HOST" "new-host"  // bool
let deleted = deleteValue store "OLD_VAR"       // bool

// Load all config at once
let allConfig = loadAll store [| "DB_HOST"; "DB_PORT"; "API_KEY" |]
// Map<string, string> with empty string for missing values

// Apply a batch of changes (handles both sets and deletes)
let results = applyChanges store [| ("DB_HOST", "old", "new"); ("OLD_VAR", "val", "") |]
// Array of (key, success, wasDelete)
```

### Interactive Editor Workflow

```fsharp
open UnionConfig.TextEditor.ConfigEditor

// Edit config interactively via $EDITOR
// 1. Loads current config into a temp .env file
// 2. Opens your editor
// 3. Diffs changes when you save and close
// 4. Runs verification on changed values
// 5. Confirms before applying
editConfig
    (fun () -> loadAll store allVarNames)
    (fun name value -> setValue store name value)
    (fun path config -> writeEnvToFile path config)
    (fun changes config -> verifyApiKeys changes config)

// Populate defaults without editor interaction
let result = populateDefaults
    (fun name -> getValue store name)
    (fun name value -> setValue store name value)
    (fun currentConfig -> getDefaultsToApply currentConfig)
    (fun config -> writeLocalEnvFile config)

match result with
| NoChangesNeeded -> printfn "All defaults already set"
| Applied count -> printfn "Applied %d defaults" count
| Failed errors -> printfn "Errors: %A" errors
```

## Reference

### Core Types

```fsharp
// How a var gets its value
type ConfigVarKind = Manual | AutoGenerated of string option | Infrastructure | AutoProvisioned

// Runtime value type (determines parsing)
type ConfigValueType = StringType | IntType | BoolType | FloatType
                     | CustomType of typeName: string * validate: (string -> string option)

// Required (fail if missing) or Optional (return None)
type ConfigRequirement = Required | Optional

// Full var definition
type ConfigVarDef = {
    Name: string; Kind: ConfigVarKind; ValueType: ConfigValueType
    Requirement: ConfigRequirement; IsSecret: bool; Doc: ConfigVarDoc }

// Parsed value
type ConfigValue = StringValue of string | IntValue of int | BoolValue of bool | FloatValue of float

// ConfigValue extraction helpers
ConfigValue.string    : ConfigValue option -> string         // required
ConfigValue.stringOption : ConfigValue option -> string option  // optional
ConfigValue.int       : ConfigValue option -> int
ConfigValue.intOption : ConfigValue option -> int option
ConfigValue.bool      : ConfigValue option -> bool
ConfigValue.boolOption : ConfigValue option -> bool option
ConfigValue.float     : ConfigValue option -> float
ConfigValue.floatOption : ConfigValue option -> float option
ConfigValue.custom    : (string -> 'T option) -> ConfigValue option -> 'T
ConfigValue.customOption : (string -> 'T option) -> ConfigValue option -> 'T option
```
